Setup Guide

Everything You Need to Get Started

Prerequisites

Windows 10/11 — Revit only runs on Windows

Autodesk Revit 2026 — Free trial at autodesk.com/revit

Node.js 18+ — For the MCP server plugin

Claude Code/Cowork — Anthropic's AI interface

$ git clone https://github.com/Demolinator/Revit-MCP-CS.git

$ cd Revit-MCP-CS

$ npm install && npm run setup

Then Start Using:

1. Open Revit 2026 and create/open a project

2. Go to Add-Ins tab, click MCP Switch

3. Open Claude Code/Cowork — ready to design!
